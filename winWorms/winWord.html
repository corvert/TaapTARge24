Windows Forms Rakendus

Kategooriate ja Toodete Halduse Süsteem

Ülevaade

See projekt on Windows Forms rakendus, mis võimaldab hallata kategooriaid ja nendele vastavaid tooteid. Rakendus kasutab SQLite andmebaasi ja on ehitatud C# keeles, kasutades Entity Framework Core'i andmebaasi operatsioonideks.

Rakendus pakub intuitiivset kasutajaliidest kategooriate ja toodete haldamiseks, võimaldades kasutajatel kiiresti ja tõhusalt hallata oma andmeid.

Funktsioonaalsus

Kategooriate haldus

  • Kuva olemasolevad kategooriad
  • Lisa uusi kategooriaid
  • Muuda olemasolevaid kategooriaid
  • Kustuta mittevajalikud kategooriad

Toodete haldus

  • Kuva kategooriaga seotud tooteid
  • Lisa uusi tooteid konkreetsesse kategooriasse
  • Muuda olemasolevate toodete andmeid
  • Seo tooted õigete kategooriatega

Andmete salvestamine

  • Automaatne andmebaasi ühenduse haldus
  • Muudatuste salvestamine "Save"-nupu abil
  • Andmete terviklikkuse tagamine
  • Veakäsitlus ja kasutaja teavitamine

Kasutatud tehnoloogiad

Keel: C# (.NET Framework)
Andmebaas: SQLite
ORM: Entity Framework Core
UI Framework: Windows Forms

Kuidas projekti käivitada

1

Eeltingimused

  • Installi Visual Studio koos .NET Framework tugi
  • Veendu, et Windows Forms App komponent on installitud
  • SQLite tugi on automaatselt kaasas
2

Projekti avamine

  • Ava .sln fail Visual Studio's
  • Lase Visual Studiol laadida kõik sõltuvused
  • Veendu, et kõik NuGet paketid on õigesti installitud
3

Rakenduse käivitamine

  • Vajuta F5 või vali menüüst "Start"
  • Rakendus loob andmebaasi automaatselt
  • Esmakordsel käivitamisel täidetakse algandmed
4

Rakenduse kasutamine

  • Kuva kategooriad vasakpoolses tabelis
  • Vali kategooria seotud toodete nägemiseks
  • Tee vajalikud muudatused
  • Salvesta muudatused "Save"-nupu abil

Andmebaasi ülesehitus

Tabel: Categories

  • Id: Unikaalne identifikaator (int, PK)
  • Name: Kategooria nimi (string)

Tabel: Products

  • Id: Unikaalne identifikaator (int, PK)
  • Name: Toote nimi (string)
  • CategoryId: Viide kategooriale (int, FK)
